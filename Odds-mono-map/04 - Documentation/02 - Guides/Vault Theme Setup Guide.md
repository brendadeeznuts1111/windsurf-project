## ğŸ” Troubleshooting

### **Theme Not Applying**
1. Check that **Minimal** theme is active
2. Verify CSS snippet is enabled
3. Restart Obsidian completely
4. Clear cache (Ctrl+Shift+R)

### **Fonts Not Loading**
1. Install Inter font on your system
2. Check font names in settings
3. Verify CSS snippet is loaded
4. Try different font variants

### **Colors Not Working**
1. Check CSS variables in snippet
2. Verify custom CSS is enabled
3. Check for conflicting styles
4. Test with different accent colors
